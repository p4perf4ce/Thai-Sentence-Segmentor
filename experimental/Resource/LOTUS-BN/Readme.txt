โครงสร้างคลังข้อมูลเสียง LOTUS-BN
all_dict.DICT 
	- พจนานุกรมคำอ่าน ของคำทั้งหมดที่อยู่ในคลังข้อมูลเสียง
- Format: 
	คำ 	สัญลักษณ์แทนเสียงอ่าน

ข้อมูลจาก VDO แต่ละรายการข่าว (session)
	ประกอบด้วย
	- session_id.wav
	- session_id.trs
	- session_id.txt
	- session_id (directory)
	ตัวอย่าง session_id 21-01-08_12-00_MCOT_01_1

wav
	ข้อมูลเสียงที่ดึงจาก VDO ของ session นั้น
trs
	xml file ที่ได้จากโปรแกรมถอดความ Transcriber ประกอบด้วยการแบ่งช่วงตามเนื้อหาข่าวและผู้พูด การกำกับข้อมูล เช่น ชนิดของข่าว ข้อมูลผู้พูด เวลาเริ่มต้นของแต่ละประโยค และข้อความที่ได้จากการถอดความ (ดูรายละเอียดวิธีการถอดความใน thai How to transcribe.doc)
txt
	ข้อมูลรายประโยค โดยดึงข้อมูลมาจาก  xml file (.trs)
directory
	 wave file ของแต่ละประโยค
